# 19 - React Form Component

Mini proyecto para practicar manejo de formularios en React.

## Objetivo

Crear un componente de formulario reutilizable que permita capturar datos del usuario.

## Funcionalidades

- Input controlado con useState
- Manejo de eventos de formulario
- Envío del formulario
- Mostrar los datos ingresados

## Conceptos que se practican

- useState
- Formularios controlados
- Eventos en React
- Manejo de inputs
- Componentes reutilizables
