# 📝 React Todo App Pro

Aplicación de gestión de tareas desarrollada con React.

## 🎯 Objetivo

Construir una Todo App completa que permita al usuario:

- Crear tareas
- Editarlas
- Eliminarlas
- Marcarlas como completadas
- Filtrarlas
- Persistir los datos en el navegador

---

## 🚀 Features

- ➕ Agregar tareas
- ❌ Eliminar tareas
- ✏️ Editar tareas
- ✅ Marcar como completadas
- 🔍 Filtros:
  - Todas
  - Activas
  - Completadas
- 📊 Contador de tareas
- 💾 Persistencia con localStorage

---

## 🧠 Conceptos a practicar

- useState
- Renderizado condicional
- Manejo de listas y keys
- Eventos (onClick, onSubmit, onChange)
- Estado derivado
- Persistencia de datos (localStorage)

---

## 📦 Requisitos

- Node.js
- React (Vite recomendado)
