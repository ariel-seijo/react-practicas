# 18 - React Modal

Mini proyecto para practicar manejo de estado y componentes de interfaz superpuestos en React.

## Objetivo

Crear un modal (ventana emergente) que se pueda abrir y cerrar mediante botones.

El modal debe aparecer por encima del contenido de la página y bloquear la interacción con el fondo.

## Funcionalidades

- Botón para abrir el modal
- Botón para cerrar el modal
- Fondo oscuro detrás del modal (overlay)
- El modal debe renderizarse condicionalmente

## Conceptos que se practican

- useState
- Renderizado condicional
- Componentes reutilizables
- Manejo de eventos
- Organización de UI en React
