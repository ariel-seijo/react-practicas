# 22 - React Error Handling

Mini proyecto para practicar manejo de errores al consumir APIs.

## Objetivo

Mostrar un mensaje de error cuando la petición a la API falla.

## Funcionalidades

- Obtener usuarios desde una API
- Mostrar un estado de carga mientras se obtienen los datos
- Mostrar un mensaje de error si la petición falla
- Mostrar la lista de usuarios si todo funciona correctamente

## Conceptos que se practican

- useState
- useEffect
- fetch
- manejo de loading
- manejo de errores
- renderizado condicional
