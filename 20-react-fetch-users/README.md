# 20 - React Fetch Users

Mini proyecto para practicar consumo de APIs en React.

## Objetivo

Obtener una lista de usuarios desde una API pública y mostrarlos en pantalla.

## Funcionalidades

- Obtener datos desde una API usando fetch
- Guardar los datos en el estado
- Renderizar una lista de usuarios
- Ejecutar la petición cuando el componente se monta

## Conceptos que se practican

- useEffect
- fetch API
- manejo de estado con datos externos
- renderizado de listas
