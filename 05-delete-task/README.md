# 05 - Eliminar tareas

Mini proyecto para mejorar la lista de tareas.

## Objetivo

Agregar la posibilidad de **eliminar tareas** de la lista.

Cada tarea mostrada debe tener un botón **Eliminar**.

Al hacer click en el botón:

- La tarea debe desaparecer de la lista.
- El estado debe actualizarse correctamente.

## Conceptos practicados

- Arrays en `useState`
- Método `filter`
- Manejo de eventos en React
- Renderizado de listas con `map`
- Actualización de estado basado en estado previo
