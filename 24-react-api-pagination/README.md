# 24 - React API Pagination

Mini proyecto para practicar consumo de APIs con paginación.

## Objetivo

Crear un componente que obtenga usuarios desde una API y permita navegar entre diferentes páginas de resultados.

## Conceptos practicados

- useState
- useEffect
- fetch API
- manejo de loading
- manejo de errores
- paginación
- renderizado de listas

## Funcionalidades

- Obtener usuarios desde una API
- Mostrar lista de usuarios
- Botón "Anterior"
- Botón "Siguiente"
- Mostrar página actual
- Deshabilitar botones cuando corresponda
- Mostrar loading mientras carga
- Mostrar error si la petición falla
