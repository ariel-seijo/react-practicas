# 23 - React API Search

Mini proyecto para practicar búsquedas dinámicas usando una API.

## Objetivo

Crear un buscador donde el usuario pueda escribir y consultar una API para obtener resultados filtrados.

## Funcionalidades

- Input de búsqueda
- Consulta a una API usando el texto ingresado
- Mostrar resultados en una lista
- Manejar loading
- Manejar errores

## Conceptos que se practican

- useState
- useEffect
- fetch
- APIs con query parameters
- inputs controlados
- renderizado condicional
