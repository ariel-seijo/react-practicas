# 03 - Input Controlado

Mini proyecto para practicar useState.

## Objetivo

Crear un input que permita escribir texto y mostrarlo en tiempo real debajo del input.

## Al escribir en el input debe:

- Mostrar lo que estás escribiendo
- Actualizarse automáticamente mientras tipeás

## Conceptos practicados

- useState
- Manejo de eventos con onChange
- Inputs controlados
- Renderizado dinámico basado en estado
