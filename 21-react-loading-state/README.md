# 21 - React Loading State

Mini proyecto para practicar manejo de estado de carga al consumir APIs.

## Objetivo

Mostrar un mensaje de carga mientras los datos se obtienen desde una API.

## Funcionalidades

- Obtener usuarios desde una API
- Mostrar un mensaje "Loading..." mientras se realiza la petición
- Mostrar la lista de usuarios cuando los datos llegan

## Conceptos que se practican

- useState
- useEffect
- fetch
- manejo de estado de carga
- renderizado condicional
