# 04 - Lista de tareas

Mini proyecto para practicar manejo de **arrays en estado**.

## Objetivo

Crear una pequeña aplicación de lista de tareas.

La app debe permitir:

- Escribir una tarea en un input
- Presionar un botón **Agregar**
- Mostrar la tarea en una lista debajo

Cada vez que se agregue una tarea nueva debe aparecer en la lista.

## Conceptos practicados

- useState
- Inputs controlados
- Arrays en estado
- Renderizado de listas con map()
- Uso de key en listas
