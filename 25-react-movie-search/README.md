# React Movie Search

Aplicación para buscar películas consumiendo una API externa.

## Objetivo

Permitir al usuario buscar películas por nombre y visualizar los resultados en tiempo real.

## Funcionalidades

- Buscar películas por título
- Mostrar lista de resultados
- Mostrar título, imagen y año
- Manejo de loading
- Manejo de errores
- Evitar búsquedas innecesarias
- UI simple y clara

## Conceptos practicados

- useState
- useEffect
- fetch API
- manejo de formularios
- renderizado condicional
- consumo de APIs
