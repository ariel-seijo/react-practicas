# 02 - React Toggle

Mini proyecto para practicar `useState`.

## Objetivo

Crear un botón que permita alternar entre dos estados.

Al hacer click en el botón debe:

- Mostrar "Encendido"
- Mostrar "Apagado"

Cada click debe cambiar entre ambos estados.

## Conceptos practicados

- useState
- Manejo de eventos en React
- Renderizado condicional
